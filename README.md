# EduAI: Personalized AI Study Assistant

## Overview

EduAI is an AI-powered educational assistant developed to support United Nations Sustainable Development Goal 4 (Quality Education).

The application helps students by providing instant answers to academic questions, concept explanations, and concise learning assistance using Google's Gemini Generative AI.

## Features

* AI Question Answering
* Concept Explanation
* Topic Summarization
* Light/Dark Theme
* Modern User Interface
* Copy Response Feature
* Responsive Design

## Technology Stack

* Python
* Flask
* HTML
* CSS
* JavaScript
* Google Gemini API

## Project Objective

To improve access to educational support by providing students with an intelligent AI-powered learning assistant.

## SDG Alignment

**SDG 4: Quality Education**

EduAI promotes accessible and personalized learning experiences through Artificial Intelligence.

## Installation

```bash
pip install flask google-genai
python app.py
```
